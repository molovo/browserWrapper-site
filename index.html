browserWrapper is a jQuery plugin, which builds a 1:1 scale responsive browser window, at any size, around any element in the DOM.

Created by @molovo and @japborst.

Features

Inspiring Iframes

Why have a plain box, when you can have a living browser window right on your web page?

Go ahead, type a URL into the address bar on your right →

The back and forward buttons switch between the last entered URLs, as the actual iframe location cannot be deciphered.

Clicking the full-screen button opens the current URL in a new tab.

Sublime Screenshots

Add life, and a touch of realism to boring webpage screenshots.

Rambunctiously Responsive

browserWrapper works in responsive layouts, meaning visitors will always see the window controls at 1:1 scale.

Look at me, I'm responsive, see me shrink!

							
  .browserWrapper({
    os: 'windows',
    browser: 'firefox',
    url: 'http://browserwrapper.molovo.co.uk',
    title: 'Options',
    favicon: '/img/favicon.ico'
  });

Completely Configurable

Tailor the browser window to match the content within, with customisable URLs, Titles and favicons.

Use It

HTML

Two elements are all that is needed. They can be any element, as long as .browser-content is within the called element, browserWrapper will do the rest.

					
<!-- Add the CSS and JS files -->
<link rel="stylesheet" href="browserWrapper/src/css/browserwrapper.css" />
<script type="text/javascript" src="browserWrapper/src/js/jquery.browserwrapper.min.js"></script>

<!-- Create your browser window -->
<div id="browser">
  <div class="browser-content">
    <!-- Your content goes here -->
  </div>
</div>

jQuery

Just call .browserWrapper() on document.ready to get started.

					
jQuery(document).ready(function ($) {
  $('#browser').browserWrapper();
});

Options

If you want to be really picky, just apply any of the following options:

					
$('#browser').browserWrapper({
  // title to appear on tab
  browserTitle:     'Your Awesome Title',

  // URL to appear in address bar
  browserURL:       '//your.awesome.url.com',

  // URL for favicon to appear on tab
  favicon:          '//browserwrapper.molovo.co.uk/img/favicon.png',

  // path to browserWrapper
  filePath:         'browserWrapper/src/',

  // adds a box-shadow to the browser, follows CSS syntax, use '0' to remove
  shadow:           '0px 5px 15px rgba(0,0,0,0.3)',

  // the OS for the window border
  os:               'mac' || 'windows' || 'detect',
                    // detect options employ user-agent detection to mimic the users browser/OS

  // if windows is selected, choose the version
  winVersion:       'windows7' || 'windows8',

  // the browser whose controls to mimick
  browser:          'chrome' || 'firefox' || 'detect' || false,
                    // false option loads the window border only with no browser controls

  // make the browser controls work with an internal iframe/object
  makeBrowserWork:  true || false,

  // enter a path to a custom theme
  theme:            'themes/lines/'
});
